# Referencias

Esta carpeta recoge referencias y notas de contexto relacionadas con la skill de cálculo asistido del Índice de Industrialización — IdI.

## Norma de referencia

La skill se ha desarrollado tomando como referencia la UNE-PNE 41610 sobre construcción industrializada de edificios, definiciones, clasificación e índice de industrialización.

Por motivos de propiedad intelectual, este repositorio no redistribuye el texto completo de la norma.

La consulta del documento normativo debe realizarse a través de los canales oficiales habilitados por UNE/AENOR u otras fuentes autorizadas.

## Criterio de uso

La skill no pretende sustituir la norma ni interpretarla de forma vinculante.

Su función es ayudar a:

- estructurar el análisis;
- identificar partidas potencialmente industrializadas;
- calcular el IdI de forma asistida;
- generar una base revisable por el consultor;
- documentar trazabilidad y criterios de validación.

La validación final corresponde siempre al técnico o consultor competente.
