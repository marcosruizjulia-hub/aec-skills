# Checklist de validación del IdI

## Antes del cálculo

- [ ] El presupuesto está completo.
- [ ] El PEM total está identificado.
- [ ] El archivo no contiene partidas duplicadas.
- [ ] Las unidades de obra tienen descripción suficiente.
- [ ] El consultor ha definido el criterio de validación.

## Durante el análisis

- [ ] Cada partida industrializada tiene una razón de inclusión.
- [ ] Las partidas dudosas están marcadas para revisión.
- [ ] Las partidas excluidas no computan en el PEM industrializado.
- [ ] La clasificación por función es coherente.
- [ ] La clasificación por geometría es coherente.

## Después del cálculo

- [ ] El IdI se ha calculado sobre el PEM total correcto.
- [ ] El PEM industrializado coincide con la suma de partidas válidas.
- [ ] El informe separa resultado automático y validación profesional.
- [ ] El consultor ha revisado las partidas relevantes.
- [ ] El informe final incluye fecha, versión y responsable.
